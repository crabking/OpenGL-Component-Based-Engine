virtual void CameraComponent::cameraQuatRotate ( float  pitch,
float  yaw 
)
pure virtual

QuatCameraRotation interface.

This function uses reverse order of the multiplications to imitate the constraints of one axis i.e. having control of one axis does not make the camera convoluted after pitching and yawing.

Parameters
pitchleft and right rotations.
yawup and down rotations.
